# BlinkIT Retail Sales Analytics Dashboard

## Project Overview

This project analyzes BlinkIT retail sales data using Excel, SQL, and Power BI.

## Tools Used

- Microsoft Excel
- MySQL
- Power BI

## Key Metrics

- Total Sales: ₹10.56M
- Average Sales: ₹2.27K
- Total Items: 4,650
- Average MRP: ₹141.72

## Key Insights

- Low Fat products generated 64% of revenue.
- Supermarket Type1 contributed 82% of total sales.
- Snack Foods emerged as the top-performing category.
- Outlet size and location significantly impacted sales performance.

## Dashboard Features

- KPI Cards
- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Fat Content
- Outlet Trend Analysis
- Interactive Slicers
